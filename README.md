## Run the application

1. Download the app: [App source link](https://drive.google.com/file/d/1utpaXgifDhdkorO1ewwOW8nnOM3-1xv8/view?usp=sharing)
   
2. Start the app:<br>
   ```
   cd [app location]
   
   docker-compose up -d
   ```

3. App will be available at:<br>
    http://localhost:3000<br>
    - Email: admin@example.com
    - Password: password
